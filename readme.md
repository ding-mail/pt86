# 2、安装方法
## 1. 将程序文件复制到PT86上
- 用U盘方式：接上U线，将所有文件复制到U盘的 /bin/ 中
- 用FTP方式：将所有文件复制到 /datafs/bin/ 中

## 2. 在PT86上依次选：系统菜单 - [3]系统设置 - [1]设定程序 - 插件 选为：virgin.main.js - 设定 - 按 ESC 到 系统菜单 - [1]运行程序
